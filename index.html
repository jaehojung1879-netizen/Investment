How it works

정적 사이트지만 실제 ML 파이프라인을 갖습니다.

config.json (추적 종목)→ GitHub Actions→ yfinance + FRED 수집→ LightGBM walk-forward + 캘리브레이션→ data/site-data.json→ 이 대시보드
이 도구는 개인 리서치·교육 목적이며 투자 조언이 아닙니다. 과거 성과는 미래를 보장하지 않으며 데이터 지연·오류 가능성이 있으므로 실제 주문 전 원천 데이터를 재확인하세요.
