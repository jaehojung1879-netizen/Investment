Market Regime

Factor Lens

QQQ/SPY 상대강도, 성장주 민감도, 금리 변화, 달러/유동성 프록시를 분리해서 봅니다.

Risk Budget

최대낙폭, 변동성 목표, 포지션 한도, 손절/재진입 조건을 대시보드 규칙으로 관리합니다.

Backtest Reality

거래비용, 슬리피지, 세금, T+1 체결, 표본 외 구간을 반영해 실행 가능한 성과만 남깁니다.

Model Workflow

노트북 분석을 서비스형 리서치 파이프라인으로 전환합니다.

  1. Data: QQQ, SPY, VIX, 2Y/10Y 금리, 수익률 곡선, 리스크 프록시를 수집합니다.
  2. Features: 모멘텀, 변동성, RSI, MACD, Bollinger, 상대강도, 금리 스트레스를 계산합니다.
  3. Model: walk-forward 학습과 isotonic calibration으로 horizon별 확률을 보정합니다.
  4. Publish: 배치 산출물을 JSON으로 저장하고 GitHub Pages 화면이 이를 읽어 표시합니다.

GitHub Pages Ready

GitHub 서버를 통해 인터넷 사이트로 확인할 수 있게 준비했습니다.

저장소를 GitHub에 push한 뒤 Pages 설정 또는 포함된 GitHub Actions 워크플로를 사용하면 https://사용자명.github.io/저장소명/ 형태로 접속할 수 있습니다.

01

Repository push

현재 브랜치를 GitHub 원격 저장소에 push합니다.

02

Pages source

Settings → Pages에서 GitHub Actions를 선택합니다.

03

Automatic deploy

.github/workflows/pages.yml이 정적 파일을 업로드하고 배포합니다.
