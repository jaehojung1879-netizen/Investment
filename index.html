How it works

정적 사이트지만 실제 분석 파이프라인을 갖습니다.

  1. 철학: docs/investment-philosophy.md에 분석 원칙과 행동 프레임워크를 기록합니다.
  2. 수집: GitHub Actions가 공개 일간 CSV를 가져옵니다.
  3. 계산: Python 스크립트가 SMA, RSI, 변동성, drawdown, QQQ/SPY 상대강도를 계산합니다.
  4. 게시: 계산 결과를 data/market-data.json으로 저장하고 Pages가 그대로 배포합니다.
  5. 해석: 브라우저는 Markdown과 JSON을 읽어 같은 화면에서 원칙과 데이터를 연결합니다.
