# 투자 원칙 · 체크리스트

## 핵심 철학

- 예측보다 **확률과 리스크 관리**를 우선한다.
- 단일 매수/매도 문구가 아니라 **시장 국면, 추세, 변동성, 상대강도, 금리 환경**을 함께 본다.
- 모델이 맞는지보다 **기준선(baseline) 대비 lift가 있는지**를 먼저 검증한다.
- 강한 상승장에서는 방어 신호가 과민할 수 있고, 하락장에서는 반등 신호가 함정일 수 있으므로 **국면별로 분리해** 해석한다.
- 실전에서는 수수료·슬리피지·세금·체결 지연을 반영한 뒤에도 의미가 있어야 한다.

## 신호를 읽는 법

1. **Multi-horizon**: 21D(단기 노이즈)·63D(핵심 추세)·126D(큰 국면)를 분리해서 본다.
2. **확률은 입력값**: 상승확률은 행동 명령이 아니라 포지션 크기와 확인 강도를 조절하는 값이다.
3. **근거 동반 확인**: 알림(STRONG BUY/SELL)은 항상 그 옆 OOS 정밀도·lift·백테스트와 함께 본다. lift가 0에 가까우면 신뢰도를 낮춘다.
4. **매크로 우선**: 금리차 역전·VIX 급등 같은 거시 경고는 개별 종목 신호보다 먼저 점검한다.
5. **국면 정합성**: 모델 신호와 리스크·국면 진단이 같은 방향일 때만 확신을 키운다.

## 행동 프레임워크

- **Constructive**: 추세·상대강도가 양호하고 거시 리스크가 낮을 때 → 핵심 비중 유지/점진 확대.
- **Neutral**: 추세는 유지되나 단기 과열·변동성·금리 부담이 혼재 → 관망 또는 분할 접근.
- **Defensive**: SMA200 하회 + 큰 낙폭 + 변동성 급등 + 상대강도 훼손이 겹칠 때 → 현금/헤지/비중 축소 검토.

## 단기 트레이드의 한계 (스스로 기억할 것)

- 단기 가격 방향은 랜덤워크에 가깝다. 확률 0.5를 조금 넘는 건 우위가 아니다 → **비용·세금을 넘는 기대값(edge)** 이 있을 때만 행동한다.
- "오늘의 트레이드"는 비용 허들과 확신 하한을 통과한 것만 제안한다. 제안이 없는 날은 **관망이 정답**이다.
- 보유 기한(hold-until)은 약속이 아니라 재평가 시점이다. 무효화 조건(MA20 이탈·확률 0.5 하회)이 먼저 오면 그때 청산한다.
- 후보를 여러 개 스크리닝할수록 우연히 좋아 보이는 게 섞인다. 상위 1~2개만, 분산해서.

## 매매 전 체크리스트

- [ ] 이 종목의 21/63/126D 신호가 서로 모순되지 않는가?
- [ ] 알림의 OOS 정밀도가 기준선을 의미 있게(>5%p) 넘는가?
- [ ] 백테스트가 Buy&Hold 대비 우위(vs B&H > 0)이고 MDD를 감당할 수 있는가?
- [ ] 매크로(금리차·VIX)가 위험 회피를 가리키고 있지는 않은가?
- [ ] 리스크·국면 진단에서 활성화된 플래그가 내 가설을 반박하지 않는가?
- [ ] 진입 시 포지션 크기와 손절·재확인 기준을 미리 정했는가?
- [ ] 지금의 행동이 단기 감정이 아니라 위 원칙에 근거하는가?
