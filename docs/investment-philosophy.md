# 투자 원칙 · 안전 체크리스트

- 현재 화면은 **모의 신호(PAPER ONLY)** 전용이며 실제 매매 지시가 아닙니다.
- `모델 점수`는 검증 미달 상태에서 상승확률로 해석하지 않습니다.
- seed, stale, 모델 0, 커버리지 부족, 품질 게이트 미통과, 생존편향/데이터 오류가 있으면 매매 사용을 금지합니다.
- 켈리 비중은 확률·payoff·상관·기존 보유비중이 검증되고 로컬 포트폴리오 상태가 제공되기 전까지 null입니다.
- 백테스트는 t일 종가 신호, t+1일 시가 체결, t+1일 종가 평가 순서로만 해석합니다.
- 유니버스는 현재 상장/구성 종목 중심이라 생존편향이 남아 있습니다.
- 매크로·나침반·공포탐욕 점수는 규칙 기반 참고 노출도이며 통계적으로 최적화된 자산배분이 아닙니다.
