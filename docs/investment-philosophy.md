# Investment Philosophy

## 핵심 철학

- 예측보다 **확률과 리스크 관리**를 우선한다.
- 단일 매수/매도 문구가 아니라 **시장 국면, 추세, 변동성, 상대강도, 금리 스트레스**를 함께 본다.
- 모델이 맞는지보다 **기준선 대비 lift가 존재하는지**를 먼저 검증한다.
- 강한 상승장에서는 방어 신호가 과민할 수 있고, 하락장에서는 반등 신호가 함정일 수 있으므로 **국면별 성능**을 분리한다.
- 실전에서는 수수료, 슬리피지, 세금, 체결 지연을 반영한 뒤에도 의미가 있어야 한다.

## 분석 원칙

1. **Multi-horizon**: 21D, 63D, 126D를 분리해 단기 노이즈와 중기 추세를 혼동하지 않는다.
2. **Trend first**: 가격이 장기 평균 위에 있고 상대강도가 우위일 때만 공격적 해석을 허용한다.
3. **Risk budget**: 변동성 확대, drawdown, 금리 스트레스가 커지면 기대수익보다 생존을 우선한다.
4. **Probabilistic view**: 확률은 행동 명령이 아니라 포지션 크기와 확인 강도를 조절하는 입력값이다.
5. **Out-of-sample discipline**: 최근 표본 외 구간에서 무너지는 신호는 대시보드에서 경고로 표시한다.

## 대시보드가 답해야 하는 질문

| 질문 | 확인 지표 |
| --- | --- |
| 지금 추세는 유효한가? | QQQ 가격, SMA50, SMA200, 63D momentum |
| 위험이 커지고 있는가? | realized volatility, VIX proxy, max drawdown |
| QQQ가 시장을 이기고 있는가? | QQQ/SPY relative strength |
| 단기 과열 또는 침체인가? | RSI14, 21D return |
| 행동해야 하는가? | composite score와 risk flag를 함께 검토 |

## 행동 프레임워크

- **Constructive**: 추세와 상대강도가 모두 양호하고 리스크 압력이 낮을 때 핵심 비중 유지 또는 점진적 확대.
- **Neutral**: 추세는 유지되지만 단기 과열, 변동성, 금리 스트레스가 혼재할 때 관망 또는 분할 접근.
- **Defensive**: SMA200 하회, 큰 drawdown, 변동성 급등, 상대강도 훼손이 겹칠 때 현금/헤지/비중 축소 검토.
